课程教案

Asp.Net Asp.Net WebForms简介

本讲对 Asp.Net 进行了简单的介绍，它是一套免费的用于制作网站或则网络应用的框架。、WebForms 是其最古老，最初始的开发模式，它是一种基于控件和事件驱动的开发模式。

Asp.Net 开发前准备

本讲介绍如何搭建 Asp.Net 的开发环境，针对 Visual Studio 的下载，以及对其界面进行了一个简单的介绍 程序，让大家对 WebForms 这种开发模式有个简单的认识。

Asp.Net MVC 简介

本讲对 Asp.Net MVC 进行了简单的介绍，MVC 作为当前开发网站最主流的开发模式，同样也是当前asp.Net 最主流的开发模式，同样在其他应用开发的时候也能见到类似的开发模式。

Asp.Net MVC 添加 Controller

本讲对 Asp.Net MVC 添加 Controller , 学习如何添加 Controlle 和 Action。

Asp.Net MVC Controller 相关

本讲对 Asp.Net MVC Controller 进行讲解，特别是路由，debug，IIS Express, HtmlEncode 等。

Asp.Net MVC 添加 View与Razor 引擎

Asp.Net MVC 添加 View与Razor 引擎

Asp.Net MVC 添加 View与Razor 引擎

本讲对 Asp.Net MVC View 进行讲解，重点解释了 Layout 的使用和如何由 Controller 向 View 传递数据。

Asp.Net MVC 添加 Model

本讲对 Asp.Net MVC 添加 Model，并创建 Entity Framework 的 Data Context。

Asp.Net MVC 连接和创建数据库 LocalDB

本讲主要讲解如何使用 Entity Framework 连接和创建数据库。主要是通过添加 web.config 中 LocalDB 的数据库连接字符串，Entity Framework 的 Code First 模式会自动连接和创建 LocalDB.

Asp.Net MVC 从页面到 Action

本讲讲解了 Asp.Net MVC 如何从页面一个链接到 Controller 的 Action ,并解释了 HttpGet 和 HttpPost 的概念。

Asp.Net MVC 解析 Edit 的 Action 和 View

本讲主要讲解 Edit 在 HttpGet 和 HttpPost 请求下的 Action 和 View 。

1. Asp.Net MVC 解析 Details 和 Delete 的 Action 和 View

本讲主要讲解 Details 和 Delete 在 HttpGet 和 HttpPost 请求下的 Action 和 View。

Asp.Net MVC 添加搜索功能

本讲讲解了 Asp.Net MVC 添加一个搜索功能，主要是通过 url 来搜索。

Asp.Net MVC 通过 UI 进行搜索

本讲主要讲解如何修改页面来添加搜索的功能。

Asp.Net MVC 基于已有种类进行搜索

本讲主要讲解 Asp.Net MVC 如何给予已有的种类型类进行搜索。

Asp.Net MVC 添加校验机制

本讲讲解了 Asp.Net MVC 添加一个校验机制，主要是通过修改 Model 的数据模型。

Asp.Net MVC 校验机制的浏览器实现效果

本讲讲解了 Asp.Net MVC 的校验机制的浏览器段，用 Javascript 的实现效果。

Asp.Net MVC 校验机制的服务器端实现效果

本讲主要讲解 Asp.Net MVC 的校验机制在禁用浏览器 Javascript 的情况下，服务器端的校验效果。
