课程教案

Java初体验

本讲主要讲解 Java 目前的应用、如何搭建 Java 开发环境、以及如何使用工具进行 Java 程序的开发，为以后的学习打下基础

变量和常量

本讲主要讲解如何在 Java 程序中应用变量和常量进行数据操作、数据类型分类及基本数据类型间的类型转换，帮助你对 Java 语言建立初步认识

常用的运算符

Java 提供了一套丰富的运算符来操纵变量。学完这一讲后您将会理解并能够灵活的使用各种运算符

流程控制语句

本讲主要讲解 Java 中的流程控制语句，包括选择结构、循环结构、跳转语句等。学完这一讲后将对程序执行流程有更深的理解，会使用流程控制语句控制程序代码执行的过程

数组

本讲主要讲解 Java 中的数组，包括数组的创建与使用、使用 Arrays 类操作数组、二维数组的使用。学完这一讲后将能够熟练使用数组进行数据的存储和操作

方法

本讲主要讲解 Java 中的方法，包括方法的定义、方法的调用、参数传递和方法重载等。学完这一讲后将对方法的概念有深层次的理解，并能够通过定义方法来实现独立的功能

类和对象

本讲内容主要讲解 Java 在面向对象编程中的基本语法，面向对象设计程序的思路

封装

封装的意思是说对象数据和操作该数据的指令都是对象自身的一部分，封装能够实现尽可能对外部世界隐藏数据

继承

继承是面向对象程序设计能够提高软件开发效率的重要原因之一

多态

多态性是允许你将父对象设置成为和一个或更多的他的子对象相等的技术，赋值之后，父对象就可以根据当前赋值给它的子对象的特性以不同的方式运作

异常与异常处理

本讲主要讲解，什么是异常、Java 中的异常体系结构、如何处理异常、如何自定义异常以及异常链的相关知识

认识 Java 中的字符串

本讲主要讲解 Java 中的字符串，字符串被广泛地用在 Java 程序设计中，很多操作都要使用字符串来完成，例如系统中存储的用户名、密码、电子邮箱等都要使用字符串来存储。学完这一讲后能够掌握字符串的处理， String 类和 StringBuilder 类常用方法的应用等...

Java 中必须了解的常用类

本讲主要讲解 Java 中常用的类，包括基本类型的包装类、Date 类、SimpleDateFormat 类、 Calendar 类、 Math 类。学完这一讲后将对基本数据类型有更深入的理解，能够进行基本类型和包装类之间的转换，掌握常用类的使用等

Java 中的集合框架

本讲主要讲解，什么是集合，Java集合框架的体系结构和集合的作用，以及常用的集合类—— ArrayList、HashSet 的讲解与实际应用。 Java集合框架中的 Map 接口及其实现类的应用 集合框架中复杂查询方法的应用、集合算法类 Collections 的应用，以及如何通过 Comparable 和 Comparator 接口进行对象的比较
